# Higgsfield 動画生成プロンプト雛形（南原竜樹 / マーケット解説）

ローカルの Claude Code（Higgsfield MCP 接続済み）にそのまま貼って使えるプロンプト集です。
`[ ]` の箇所を埋めてから実行してください。生成後は `media/` に保存し、`python src/main.py --media media/xxx.mp4` で X に投稿できます。

---

## 1. 基本テンプレート（穴埋め式）

```
Higgsfield で「南原竜樹」のマーケット解説ショート動画を作って。

- 構成スタイル: ElevenLabs 風（ナレーション音声 + 人物アバターが正面で話す解説）
- 縦横比: [9:16（縦型ショート） / 16:9（横型）]
- 尺: [約15〜30秒]
- ナレーション原稿（日本語・落ち着いた男性ボイス）:
  「[ここに話させたいセリフを入れる]」
- 画面: [テロップあり / 背景はオフィス or マーケットチャート / 落ち着いたトーン]
- 参照する顔写真: [人物の見た目を再現する場合は画像パスを指定。なければ汎用の男性ナレーター]

生成できたら media/nambara_[slot].mp4 として保存して。
```

---

## 2. 記入済みサンプル（朝の挨拶 / slot1）

```
Higgsfield で「南原竜樹」のマーケット解説ショート動画を作って。

- 構成スタイル: ElevenLabs 風（ナレーション音声 + 人物アバターが正面で話す解説）
- 縦横比: 9:16（縦型ショート）
- 尺: 約20秒
- ナレーション原稿（日本語・落ち着いた男性ボイス）:
  「おはようございます、南原竜樹です。本日のマーケットをチェックしていきましょう。
   今朝の注目は為替と米国株の流れ。詳しくはこの後の投稿で解説します。」
- 画面: 下部に大きめテロップ、背景はマーケットチャートを薄く表示、落ち着いた青基調
- 参照する顔写真: media/face.jpg（用意があれば）

生成できたら media/nambara_slot1.mp4 として保存して。
```

---

## メモ

- **ElevenLabs 風の解釈**: ElevenLabs は音声合成（TTS）が主力。ここでは「自然な日本語ナレーション + 喋るアバター」の構成を指す前提。Higgsfield 側では動画モデル（Veo / Kling / Soul 等）＋音声を組み合わせて生成します。
- **人物の見た目を再現したい場合**は、必ず参照用の顔写真を渡してください（image-to-video / アバター生成）。
- うまく出ない時は、尺・縦横比・声色・テロップ有無・背景を具体的に伝えるほど安定します。
- 台本のストックは `prompts/narration_samples.md` を参照。
